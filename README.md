# RocketsAndDragons

```sh
$ yarn install
```

```sh
$ yarn start
```

## Aim

An app to provide an overview of SpaceX's rockets and dragons.

## Features

- Users can fetch and render a list of rockets & dragons from SpaceX's API.
- Show extended information about a selected list item.

## Tech

- React
- TypeScript
- Redux Hooks
- Styled Componetns
- Webpack
